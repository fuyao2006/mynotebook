====== 面向对象1 ======
创建日期 星期三 11 十月 2017

类定义
[修饰符] class 类名
{
零个到多个构造器
零个到多个变量
零个多多个方法
}
修饰符：public final abstract
 变量定义
[修饰符] 类型 变量名[=默认值] 
修饰符：public protected private static final 

方法定义
[修饰符] 类型 方法名（形参列表）
修饰符：public protected private static final  abstract

构造器
[修饰符] 构造器名（形参列表）
修饰符：public protected private
